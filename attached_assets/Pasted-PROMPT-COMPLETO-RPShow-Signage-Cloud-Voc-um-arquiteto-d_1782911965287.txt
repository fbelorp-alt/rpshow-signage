PROMPT COMPLETO – RPShow Signage Cloud

Você é um arquiteto de software sênior, especialista em SaaS, ERP,
Digital Signage, UX/UI, infraestrutura cloud e desenvolvimento Full
Stack.

Crie um sistema web moderno chamado RPShow Signage Cloud, com aparência
premium, tema escuro, interface responsiva e arquitetura preparada para
milhares de clientes e telas.

Objetivo

Desenvolver uma plataforma completa para gerenciamento de Digital
Signage, Painéis de LED, TVs corporativas e players
Android/Windows/Linux.

O sistema deve permitir:

-   Cadastro de clientes
-   Cadastro de telas
-   Cadastro de players
-   Monitoramento em tempo real
-   Controle financeiro
-   Emissão de boletos
-   Cobrança recorrente
-   Contratos
-   Ordem de Serviço
-   Chamados
-   Gestão de conteúdo
-   Playlists
-   Agendamento
-   Dashboard executivo
-   Relatórios
-   API REST
-   Multiempresa (SaaS)
-   Controle de permissões
-   Assistente IA
